# 프로그램 UPDATE 내역

### 수동설치 방법
```yml
이전 버전으로 돌아가기 : 

1. 원하는 버전의 update 파일을 다운로드 받는다
2. 다운로드 받은 파일을 압축 해제한다.
3. 압축 해제 한 파일들을 C:\DentalLab 안에 덮어쓰기 한다.
```
